Linux:
1. create a general executable:
"pyinstaller --onefile --windowed VMM.py"  

2.Prepare for conversion to AppImage:
"mkdir -p AppDir/usr/bin"
"cp -r dist/* AppDir/usr/bin/"


3. Create AppRun file in AppDir(created in previous step):

#!/bin/bash
HERE="$(dirname "$(readlink -f "$0")")"
export PATH="$HERE/usr/bin:$PATH"
exec "$HERE/usr/bin/VMM" "$@"

4. Make the AppRun file executable:
"chmod +x AppDir/AppRun"

5. Make the AppImage:
"appimagetool AppDir"



Windows:
Please note you will need to install python 3.13.7 and PyInstaller to make this work, nothing else should be required(I dont use windows anymore)

1. Copy folder to another location (Optional)
"cp -r ./VMM_Folder ./VMM_Folder_Windows/"

2. Create the exe:
Using Wine:
'wine PyInstaller --onefile --windowed --hidden-import=PIL  "Z:\Path\To\VMM.py"'
Using Windows:
'PyInstaller --onefile --windowed --hidden-import=PIL  "Z:\Path\To\VMM.py"'
